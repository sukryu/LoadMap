# 개발자 로드맵

## 1. 컴퓨터 사이언스(CS) 기본기 다지기
### 1.1 데이터 구조와 알고리즘
- **자료구조**:
  - 배열, 연결 리스트, 스택, 큐, 트리, 그래프, 해시 테이블, 힙, 색인 순차 파일 등.
- **알고리즘**:
    1. 기초 알고리즘
        1. 정렬 알고리즘
            - 선택 정렬, 삽입 정렬, 버블 정렬, 셸 정렬
        
        2. 탐색 알고리즘
            - 선형 탐색, 이진 탐색, 해시 탐색

        3. 재귀 알고리즘
            - 일반 재귀, 꼬리 재귀, 직접 재귀, 간접 재귀, 순수 재귀

    2. 완전 탐색 (Exhaustive Search)
        1. 브루트 포스 (Brute Force)
            - 순열 탐색, 조합 탐색

        2. 백트래킹 (Backtracking)
            - N-Queens 문제, 그래프 색칠 문제

        3. 비트마스크 탐색 (Bitmasking Search)
        
    3. 비선형 구조 탐색
        - 트리 및 그래프 탐색
            - DFS (깊이 우선 탐색)
            - BFS (넓이 우선 탐색)

    4. 고급 탐색 및 최적화
        1. 효율적인 탐색
            - 점프 탐색, 피보나치 탐색, 지수 탐색

        2. 문자열 탐색
            - KMP, 라빈-카프, 보이어-무어
        
        3. 트라이 및 고급 문자열 검색
            - 아코-코라식, Z 알고리즘

    5. 다이나믹 프로그래밍 (Dynamic Programming)
        1. 기본 기법
            - 탑다운 방식, 바텀업 방식

        2. 문제 유형
            - 배낭 문제, LCS, LIS, 행렬 체인 곱셈, 에디트 거리

    6. 그리디 알고리즘
        1. 기초 그리디
            - 순수 그리디, 직교 그리디, 완화된 그리디

        2. 문제 유형
            - 허프만 코딩, 다익스트라 알고리즘, 크루스칼 알고리즘, 프림 알고리즘

    7. 그래프 알고리즘
        1. 최단 경로
            - 다 익스트라, 벨만-포드, 플로이드-워셜

        2. 최소 스패닝 트리
            - 크루스칼, 프림

        3. 네트워크 및 순서
            - 포드-풀커슨, 타잔 알고리즘, 토포로지 정렬

    8. 기타 알고리즘
        1. 트리 및 데이터 구조
            - 세그먼트 트리, 펜윅 트리, 유니언-파인드

        2. 기하 알고리즘
            - 컨벡스 헐, 최근접 쌍 문제, 보로노이 다이어그램

        3. 확률 기반
            - 몬테카를로, 라스베가스 알고리즘

        4. 분할 기법
            - 수평/수직 분할, 시뮬레이션

  1. 정렬 알고리즘
       
     2. 고급 정렬
      - 병합 정렬 (Merge Sort)
      - 퀵 정렬 (Quick Sort)
      - 힙 정렬 (Heap Sort)
      - 기수 정렬 (Radix Sort)
       
    3. 추가적인 알고리즘
      - 외부 정렬 (External Sorting, K-way Merge Sort)
      - 버킷 정렬 (Bucket Sort)
      - 카운팅 정렬 (Counting Sort)
      - 분포 정렬 (Distribution Sort, American Flag Sort)

  2. 탐색 알고리즘
    2. 트리 탐색
      - 이진 트리 탐색 (Binary Tree Search)
      - 깊이 우선 탐색 (DFS, Depth First Search)
      - 넓이 우선 탐색 (BFS, Breadth First Search)

    3. 고급 탐색
      - 점프 탐색 (Jump Search)
      - 피보나치 탐색 (Fibonacci Search)
      - 지수 탐색 (Exponential Search)
      - Z 탐색 (Z - Algorithm, 문자열 검색)

  3. 재귀 알고리즘
    2. 재귀 활용
      - 분할 정복 (Divide and Conquer)
        - 스트라센 알고리즘 (Strassen Algorithm, 행렬 곱셈 최적화)
        - 카라츠바 알고리즘 (Karatsuba Algorithm, 큰 정수 곱셈 최적화)

  4. 다이나믹 프로그래밍 (Dynamic Programming)

    2. 문제 유형
      1. 배낭 문제 (Knapsack Problem)
        - 0/1 배낭 문제, 부분 배낭 문제, 다차원 배낭 문제
      6. 최적 이진 검색 트리 (Optimal Binary Search Tree)

  5. 그리디 알고리즘
    2. 문제 유형
      1. 매칭 추구 (Matching Pursuit)
      6. 활동 선택 문제 (Activity Selection Problem)
      7. 인터벌 스케쥴링 (Interval Scheduling)

  6. 그래프 알고리즘
      - 에드몬드-카프 알고리즘 (Edmonds-Karp, BFS 기반 최대 유량)

  8. 기타 알고리즘

    2. 최적화 기법
      - 비트마스크 (Bitmasking)
      - 행렬 제곱 (Matrix Exponentiation)

    3. 기하 알고리즘
      - 컨벡스 헐 (Convex Hull)
        - 그레이엄 스캔 (Graham Scan)
        - 자비스 행진 (Jarvis March)

    5. 분할 기법
      - 수평 분할 (Horizontal Partition)
      - 수직 분할 (Vertical Partition)
      - 백트래킹 (Backtracking)
      - 브루트포스 (Brute Force)
      - 시뮬레이션 (Simulation)      

### 1.2 운영체제와 컴퓨터 네트워크
