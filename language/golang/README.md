## 백엔드 프레임워크

### Gin

* 장점:
    - 가벼운 라우팅 엔진 (httprouter 기반)
    - 빠른 성능과 간단한 API
    - 미들웨어 체인, 그룹 라우팅 등 편의 기능
    - JSON 바인딩, 인증 등에 필요한 기본 기능 제공

* 단점:
    - 완전한 MVC 프레임워크는 아니며, 필요한 기능은 플러그인 형태로 추가해야 함
